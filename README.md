# End of Year CPT Project
Incremental-Escape-Room

Game Name: Blackout Escape
# Game Goals
- Start with a blank screen with a set amount of turns
- Turns: A Resource that ticks down by 1 every 30 seconds
- Use turns to search locations, ignite lamps and solve puzzles
- Inventory system to store equipment
- Solve the puzzles with items you gather
- Skill Checks required to progress and cost turns to do
- Move from room to room to progress
- Loss Condition: Running Out of Turns. Win Condition: Escape

Game Level: 
The game will feature relatively simple puzzles with a Y6 grade difficulty

The game is very similar in essence to the board game "The Night Cage"

Game Progression: 
Starting Turns: 150
 Each turn = 30 seconds of game time AND one action point
 Target main path: ~will take about 90 turns worth of actions and 25 minutes (50 more turns) → player ends with 10 turns if efficient

Room 1: 
Dark Cellar
Turn on lamp → reveals crate & barrel → turns -1
Inspect crate → rusty key → turns -1
Inspect barrel → folded note → turns -1
Inspect shelves → dusty journal → turns -1
Open journal → find symbol clue → turns -1
Solve mini-puzzle on floorboards → turns -3
Unlock door → move to Room 2 → turns -1

Optional / Hidden (~10 turns)
Hidden drawer → token → turns -1

Scratched symbols → turns -1
Flick lamp in pattern → Easter egg → turns -2
Examine corners → hidden object → turns -1
Inspect cracks → secret map → turns -1

Risk / Fail (~3 turns)
Wait too long → lose extra turn
Trap (floorboard creak) → lose 1 turn
Drop lamp → lose 1 turn


Room 2: Abandoned Library
Theme: Riddles

Inspect bookshelves → riddle note → turns -1
Examine desk → magnifying glass → turns -1
Solve multi-step riddle → find hatch combo → turns -3
Examine additional shelves → numeric clues → turns -2
Highlight hidden letters → turns -3
Open hatch → move to Room 3 → turns -1

Optional / Hidden (~10 turns)
Hidden drawer → amulet → turns -1

Rearrange books → secret passage → turns -3
Inspect maps → clue for Room 5 → turns -2
Hidden letter codes → turns -2
Open cabinet → secret item → turns -2


# ASCII Art
The art within the game will be primarily dark with a heavy use of the characters: @, #, $, 8, B, %, W, or M
The role of this is to highlight the brightness of the lights that appear. 
This art will change as the timer progress and more lights ignite to switch to a brighter colour scheme
