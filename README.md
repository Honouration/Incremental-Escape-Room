# End of Year CPT Project
Incremental-Escape-Room

Game Name: Blackout Escape
# Game Goals
- Start with a blank screen with a set amount of turns
- Turns: A Resource that ticks down by 1 every 30 seconds
- Use turns to search locations, ignite lamps and solve puzzles
- Inventory system to store equipment
- Solve the puzzles with items you gather 
- Move from room to room to progress
- Loss Condition: Running Out of Turns. Win Condition: Escape

Game Level: 
The game will feature relatively simple puzzles with a Y6 grade difficulty

The game is very similar in essence to the board game "The Night Cage"

# ASCII Art
The art within the game will be primarily dark with a heavy use of the characters: @, #, $, 8, B, %, W, or M
The role of this is to highlight the brightness of the lights that appear. 
This art will change as the timer progress and more lights ignite to switch to a brighter colour scheme
